About the project

This project was conducted by Ann-Kristin Bergmann, Olav Seim, Torkel Westby, and Kevin Xu as part of the 2022 edition of the class "Applied Data Analysis" (CS-401) at EPFL. The project received the best project award after being identified as one of the top 5 out of 121 projects (top ~4%).

The project's code is available on GitHub.

For more information about the CS-401 class, please see the syllabus.
